# Contributing

Thanks for your interest in YouTubeTranscript.dev! The API is proprietary, but this repo is our community hub for feedback, bug reports, and feature requests.

## How You Can Help

### 🐛 Report Bugs

Found something broken? [Open an issue](../../issues) with:

1. What you were trying to do
2. The request you sent (redact your API key!)
3. The response you got
4. What you expected instead

### 💡 Request Features

Have an idea for the API? [Start a discussion](../../discussions) and tell us:

1. What problem you're trying to solve
2. How you'd ideally want it to work
3. Your current workaround (if any)

### 📖 Report Documentation Errors

Spot a mistake in the README, examples, or API docs? Open an issue and let us know.

### ⭐ Star This Repo

The simplest way to support us — helps other developers find the project.

## What's Not in Scope

This repo doesn't contain the API source code — it's a proprietary service. We don't accept code contributions to the API itself, but we absolutely value your feedback in shaping it.
